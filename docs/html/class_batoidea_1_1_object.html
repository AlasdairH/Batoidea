An object in the scene.

The object class represents a 3D object comprised of triangles which an overall bounding box.

bool Batoidea::Object::boundIntersection ( const Ray &  _ray)
inline

Tests a ray against the object's bounding box.

Parameters
_rayThe ray to test
Returns
Whether the ray intersects the objects bounds

Tests a ray against the boudning box of the object. This is used to optimise the tracing of the scene by testing to see if the ray intersects the object at all before testing each individual triangle. While this isn't perfect, it is a huge improvement in render times.
